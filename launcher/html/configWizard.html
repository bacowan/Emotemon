Steps to run

You will have to preform these steps every time you run Emotemon

Acquire required software

Make sure that you have the latest version of mGBA and a copy of Pokemon Fire Red Version (U) (V1.1).

Run mGBA and associated software

  1. Run mGBA
  2. Load the FireRed ROM
  3. Load the lua script
    1. Go to Tools -> Scripting...
    2. In the window that pops open, go to File -> Load Script...
    3. Navigate to

      and click Open
    4. Click Run in the main page of this app
